77 // Unlike phase control (which distorts every cycle identically and creates standard odd harmonics like 3rd, 5th, 7th), cycle stealing creates sub-harmonics
78 // and inter-harmonics (frequencies below 50/60Hz or between standard multiples).
79 // Since the algorithm uses a dynamic Delta-Sigma modulator (Bresenham-like) rather than a fixed pattern length, the "period" of the repetition is not fixed
80 // or easily predicted without a complex simulation.
