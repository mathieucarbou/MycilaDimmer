# Examples

The library includes comprehensive examples covering all dimmer types and use cases.

## Available Examples

| Example                                                                                                            | Description                                             |
| :----------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------ |
| [CycleStealing](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/CycleStealing/)                   | Cycle stealing dimmer with a standard Zero-Cross SSR    |
| [CycleStealingWithZCD](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/CycleStealingWithZCD/)     | Cycle stealing with a Random SSR and external ZCD input |
| [DAC](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/DAC/)                                       | DFRobot DAC module control                              |
| [DACAuto](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/DACAuto/)                               | DFRobot DAC with automatic configuration                |
| [Json](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/Json/)                                     | JSON telemetry integration                              |
| [MultipleDimmers](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/MultipleDimmers/)               | Running multiple dimmer instances simultaneously        |
| [PWM](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/PWM/)                                       | Basic PWM dimming output                                |
| [Thyristor](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/Thyristor/)                           | TRIAC control with zero-cross detection                 |
| [ThyristorAutoFrequency](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/ThyristorAutoFrequency/) | Automatic grid frequency and semi-period detection      |
| [ThyristorWithFS](https://github.com/mathieucarbou/MycilaDimmer/tree/main/examples/ThyristorWithFS/)               | File system integration for persistent settings         |
